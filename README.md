# Python Bookbot

A small command-line tool written in Python that analyzes a text file and prints a report with:

- total word count
- character frequency counts

This project was built as part of my Boot.dev coursework to practice Python fundamentals, file I/O, functions, dictionaries, sorting, and command-line arguments.

## Features

- Reads a `.txt` file from a path provided in the command line
- Counts the total number of words
- Counts how often each character appears
- Sorts and displays character counts in a readable report

## Usage

Run the program with a path to a book or text file:

```bash
python3 main.py books/frankenstein.txt
python3 main.py books/mobydick.txt
python3 main.py books/prideandprejudice.txt
